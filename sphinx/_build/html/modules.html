class pyvalid.validators.IterableValidator(**func_kwargs)[source]¶

Bases: pyvalid.validators.__base.AbstractValidator

This class performs validation on the content of the iterable to check if the given iterable is valid or not. The iterable can be either a list, tuple or even keys or values of a dictionary.

Example:

validator = IterableValidator(
    empty_allowed=False, elements_type=int, min_val=0, max_val=50
)
@accepts(validator)
def example([1, 3, 7, 10]):
    pass
iterable_types = (<class 'list'>, <class 'tuple'>, <class 'dict'>, <class 'set'>)¶
